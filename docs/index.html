ManeuverGPT

Agentic Control for Safe Autonomous Stunt Maneuvers

Paper (arXiv) YouTube Demos Code

Overview

ManeuverGPT is an agentic framework that generates and executes high-dynamic stunt maneuvers in autonomous vehicles. It combines a multi-agent LLM controller with safety validation and CARLA-based evaluation.

  • LLM agents: query enrichment, driver, validator
  • Prompted J-turn generation with safety checks
  • CARLA v0.9.14 evaluation pipeline

Autonomous Pipeline

Run the full pipeline by launching the orchestrator and CARLA client against the same Redis queue. 

export OPENAI_API_KEY="..."
python -m maneuvergpt.carla.orchestrator \
  --mode online \
  --iterations 5 \
  --redis_queue maneuver_queue \
  --user_input "Generate a safe J-turn in an open lot"

python -m maneuvergpt.carla.drive \
  --mode online \
  --redis_queue maneuver_queue \
  --sync

Offline Execution

Replay a validated maneuver directly from a JSON file.

python -m maneuvergpt.carla.maneuvers \
  --mode offline \
  --params maneuver_outputs/iteration_1/validated_maneuver_1.json
